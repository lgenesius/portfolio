Portfolios

My Recent Projects

EcoList

EcoList is an app that helps users easily create meal plan, integrate shopping list, and track their groceries at home all at once.

  • Written in Swift and built the iOS app's UI using UIKit with XIB components.

  • Supports multi-platform integration which has watchOS companion app built using SwiftUI.

  • Supports accessibility features which is dark mode and large text accessibility.

  • Dynamic height of UITableView and UITableViewCell.

  • Architecture pattern mainly using MVVM.

  • Frameworks used are UIKit, SwiftUI, CoreData, UserNotifications, and WatchConnectivity.

App Store Video Demo

Myres

Myres is a photo album app but each photo have a story. With this, you can save your personal story easier. Myres also can add new photo from photo library or directly take picture using camera from Myres.

  • Written in Swift and built the UI using UIKit with Storyboard.

  • Support dark mode.

  • Data saved in offline storage using CoreData.

  • Image saved in document directory using FileManager.

  • Implemented responsive UICollectionView in all iPhone devices using auto layout.

  • Architecture pattern using MVC.

  • First app published to the App Store.

  • Frameworks used are UIKit, CoreData, AVFoundation, Photos, and PhotosUI.

App Store Video Demo

Pick-A-Pic!

Pick-A-Pick! is an app to help online shop owner take better photos by providing guidance to understand better lighting and angle as well as better way to organize those photos.

  • Written in Swift and built the UI using UIKit with Storyboard.

  • Support dark mode.

  • Delivered angle mechanism feature using CoreMotion.

  • Delivered custom camera using UIKit programmatically and AVFoundation framework.

  • Architecture pattern using HMVC.

  • Frameworks used are UIKit, CoreData, CoreMotion, ARKit and RealityKit.

Video Demo

Edukata

Edukata is an iOS application to help dyslesic childrens to improve their reading ability by introducing words. Edukata used multi-sensory method in the form of a game with bright visual color and music to make it more interactive so childrens don't get bored in learning.

  • Written in Swift and built the UI using UIKit with Storyboard.

  • Voice guidance using AVSpeechSynthesizer.

  • Drag and drop mechanism using UIDragInteraction and UIDropInteraction.

  • Celebration using CAEmitterLayer and hand animation guide using CABasicAnimation.

  • Architecture pattern using MVC.

  • Frameworks used are UIKit, CoreData and AVFoundation.
